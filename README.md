초기화면 -> 브랜드 , 가격 , 차량 종류 등 한 눈에 들어갈 수 있게 할 것

어드민 화면 -> 광고 화면 수정 및 판매 매출 등 확인 할 수 있게 할 것

저장된 견적 -> 본인이 짠 견적 보여줄 것

CBTI 질문 -> 말 그대로 질문지 간단하게 5~6개정도 준비

질문을 통한 추천 견적 -> CBTI를 통해서 준비한 견적 출력
